# 强化学习

#### 介绍
本项目为个人强化学习的学习代码，算法包括`Q-Learning`、`DQN`、`DDQN`、`PolicyGradient`、`ActorCritic`、`DDPG`、`PPO`、`TD3`、`SAC`。

#### 使用说明

- **python版本：**` 3.10.13`

- **依赖库：**[requirements.txt](./requirements.txt)

- **安装依赖库：**`pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple/`
